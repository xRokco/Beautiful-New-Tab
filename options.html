Changelog

    1.9

  • Fixed the other bookmarks check so it actually checks that you actually have bookmarks in the folder rather than just checking that the folder exists
  • Improved some responsiveness on narrow screens/half screen chrome layouts

    • 1.8

  • Changed how the text fade out worked to hide text that is too long completely now and better match Chrome
  • Added a check to make sure the user actually has other bookmarks before creating the folder
  • changed some colours and font size to try better match Chrome on Windows look
  • Fixed the Other bookmarks background colour on hover
  • Added this changelog
  • Added labels to some setting that were missing them

    • 1.7

  • Improvements to the bookmark bar including first level folder support and other bookmarks folder support. Still many limitations such as if the user has too many long bookmarks they will just be hidden with no way to see the hidden ones yet
  • Changed some bookmark bar styling to look much closer to Chrome's

    • 1.6

  • Made it so if the box of links is left completely empty then the box that contains them won't even try to render
  • Fixed some minor typos
  • Added a primitive bookmark title cutoff length

    • 1.5

  • Added initial bookmark bar replacement - this is barely functioning at this time but does give a list of your bookmark bar bookmarks across the top of the page. No folder support yet.

    • 1.4

  • Minor bug fixes

    • 1.3

  • Changed encoding for the degree symbol as it was not rendering correctly on some machines
  • Added Kelvin option for if you live somewhere close to absolute zero.

    • 1.2

  • Changed how the scripts worked so the js file wouldnt load with every new tab
  • If an image fails to cache the extension will now retry quetly in the background, this should mean you almost never see one of the three offline images

    • 1.1

  • Added options page
  • Fixed random number generation
  • Changed the offline backup images
  • Changed the names and descriptions of the extension
  • Added option of show all images rather than just featured ones
